# Hytale Generators Pack Starter

---

This asset pack starter uses the npm package [hytale-generators](https://www.npmjs.com/package/hytale-generators).

This will allow you to generate asset pack JSON files, lang updates and colored tints of greyscale mask textures with TypeScript. The package source can be found [here](https://github.com/dustinheisey/Hytale-Generators/tree/main).
